Jakarta Project: MAILER JSP Tag Library Revision History

Version .8 released November 21, 2000

first version released for scrutiny.

Version .9 released November 22, 2000

change in the build, and a few changes made to the documentation

Version 1.0 released March 15, 2000

Support added for Jndi lookup of Session and MimePartDataSource object. Message is sent by a thread.

Version 1.1 released March 27, 2000

To, Cc and Bcc tags changed to a set and add version to allow the addition of a single address to an already created list.  Send and error tags added to make it possible for wrongly set addresses to be dealt with by the page author.

Version 1.2 released May 21, 2001

Support added for sending attachments in the mail.
